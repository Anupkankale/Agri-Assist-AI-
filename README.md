

# 🌾 AgriAssist AI

AI-powered assistant that helps Indian farmers discover government schemes
they qualify for, get weekly crop advisories, and receive location-based
fertilizer guidance.

## Tech Stack
- Frontend: Nuxt.js 3 + Tailwind CSS
- Backend: PHP Laravel
- AI Model: Nemotron-3-Super (via OpenRouter free tier)
- Vector DB: Qdrant
- Database: MySQL + Redis

## Run

```bash
cd frontend
npm install
npm run dev
```
